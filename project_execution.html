OMNI-Mesh — universal polymorphic data mesh

One codebase, three domains. A single data-mesh skeleton (Apache Iceberg + dbt medallion + Dagster + Agentic RAG) runs as MANUFACTURING, HEALTH_TECH, or ROBOTICS — selected by the OMNI_MESH_PROFILE environment variable. Consolidates the formerly separate MFG-Mesh, heal-mesh, and RoboMesh repos.

🧩 OMNI-Mesh — 🏗️ MANUFACTURING · 🩺 HEALTH_TECH · 🤖 ROBOTICS
